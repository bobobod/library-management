#### library-management

> 基于RFID的图书借阅管理系统，通过运用RFID（无线射频技术）到图书管理中，实现自助借阅的功能。

**本系统主要实现的功能有：**

* 管理员后台系统
  - 图书管理
  - 类别管理
  - 读者智能卡管理
  - 借阅管理
* 读者前台系统
  - 图书查询
  - 自助借阅/归还
  - 读者门户

**本系统主要采用的技术有：**

前端：Vue.js、ElementUI

后端：SpringBoot、MybatisPlus、SpringSecurity、Apache POI、Swagger2、aliyun短信服务
